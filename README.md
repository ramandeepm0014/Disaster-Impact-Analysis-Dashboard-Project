# 🌍 Disaster Impact Analysis Website

A professional **Python Flask-based Disaster Impact Analysis Dashboard** built using a cleaned disaster impact Excel dataset. The dashboard provides interactive visualizations, KPIs, filters, maps, and data analysis to help users understand the impact of disasters across different regions.


## 📌 Project Overview

The Disaster Impact Analysis Website is an interactive web application developed using **Flask, Python, HTML, CSS, JavaScript, and Excel**. It allows users to analyze disaster-related data through professional dashboards, dynamic charts, filters, and location-based visualization.


## ✨ Features

- 📊 KPI Dashboard
  - Total Deaths
  - Total Injured
  - Total Affected Population
  - Economic Damage
  - Relief Funds
  - Recovery Days
  - Coverage Percentage

- 🌍 Country, City, Disaster Type, Severity, and Year Range Filters

- 📈 Interactive Charts using Chart.js

- 🗺️ Disaster Location Map using Leaflet

- 📋 Filtered Dataset Table

- 🎨 Professional Responsive Dark UI

- 📂 Excel Dataset Integration

- ⚡ Fast Flask Web Application

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Disaster-Impact-Analysis-Website.git
```

### 2. Open the Project Folder

```bash
cd Disaster-Impact-Analysis-Website
```

### 3. Create a Virtual Environment

```bash
python -m venv venv
```

### 4. Activate the Virtual Environment

#### Windows (PowerShell)

```bash
venv\Scripts\activate
```

### 5. Install Required Packages

```bash
pip install -r requirements.txt
```

### 6. Run the Flask Application

```bash
python app.py
```

### 7. Open in Browser

```
http://127.0.0.1:5000
```
